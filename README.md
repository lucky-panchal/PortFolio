## 🚀 **Quick Start**

```bash
# Clone the magic
git clone https://github.com/lucky-panchal/Portfolio.git

# Enter the portal
cd Portfolio

# Install dependencies
npm install

# Launch the experience
npm start

# Build for production
npm run build
```
